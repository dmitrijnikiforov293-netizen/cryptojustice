Как заморозить средства на Binance и других биржах

Если деньги, которые у вас украли, ушли на крупную биржу — их можно заморозить. Биржи обязаны сотрудничать с правоохранительными органами. Вот как это работает на практике.

Почему биржи идут навстречу

Крупные централизованные биржи работают в регулируемой среде. Они обязаны соблюдать AML/KYC-требования и сотрудничать с правоохранительными органами. Несоблюдение грозит потерей лицензии.

При поступлении обоснованного запроса с доказательной базой биржа может: заморозить аккаунт мошенника, предоставить KYC-данные (имя, документы, IP-адреса) по запросу полиции, заблокировать вывод конкретных средств.

Важно: всё это работает только если средства ещё на бирже. Мошенники стараются вывести деньги быстро — именно поэтому первые 48 часов после обнаружения мошенничества критичны.

Сначала нужно найти средства

Прежде чем подавать запрос на заморозку — нужно доказать, что именно эти средства, на именно этом аккаунте биржи, являются вашими украденными деньгами. Для этого нужен блокчейн-анализ.

Аналитик строит цепочку: ваш кошелёк → адреса мошенника → промежуточные кошельки → горячий кошелёк биржи. Этот отчёт становится главным доказательством в запросе.

Процесс для Binance

  • 01 //
    Подготовьте блокчейн-отчёт с цепочкой транзакций, ведущих к Binance-адресу
  • 02 //
    Подайте запрос через Law Enforcement Portal Binance — law-enforcement.binance.com. Требует верификации как официальный орган, либо через партнёра.
  • 03 //
    Альтернатива — через compliance@binance.com с полным пакетом документов. Менее эффективно, но доступно частным лицам.
  • 04 //
    Ожидание ответа — обычно 2–8 недель. Binance подтверждает получение, но не всегда сообщает о результате сразу.
  • 05 //
    При положительном ответе — средства замораживаются, для возврата может потребоваться судебное решение.

Другие биржи

Kraken: принимает запросы через law enforcement portal. Известна оперативным ответом при наличии полицейского номера дела.

OKX: compliance@okx.com. Работает с запросами через юридические письма. Требует нотариально заверенных документов для крупных сумм.

Bybit: support.bybit.com — раздел для правоохранительных органов. Работает оперативно при наличии доказательной базы.

Coinbase: имеет отдельный Law Enforcement Request Portal. Одна из самых отзывчивых бирж при работе с правоохранителями.

Какие документы нужны для запроса

Блокчейн-отчёт (PDF) Цепочка транзакций Заявление в полицию Запрос на английском Доказательства потерь
  • Blockchain trace report — технический отчёт с визуализацией цепочки транзакций
  • Police report number — номер полицейского дела из вашей страны
  • Official freeze request letter — на английском языке с конкретными адресами и суммами
  • Transaction proof — скриншоты и hash транзакций из вашего кошелька
  • Identity documents — паспорт или другое удостоверение личности заявителя

Сроки и что ожидать

Реалистичные ожидания по срокам:

  • Блокчейн-анализ: 3–7 рабочих дней
  • Подготовка документов: 5–10 рабочих дней
  • Ответ от биржи: 2–8 недель
  • Заморозка (если одобрено): немедленно после решения биржи
  • Судебный возврат: 3–12 месяцев в зависимости от юрисдикции
Не ждите: каждый день промедления снижает шансы. Мошенники регулярно выводят средства с бирж. По нашей статистике, при обращении в первые 48 часов заморозка удаётся в 74% случаев. При обращении через неделю — в 41%.
